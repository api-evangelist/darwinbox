# Darwinbox

Darwinbox is a cloud-based HRMS (Human Resource Management System) platform offering a REST API for managing employee records, recruitment, performance, payroll, attendance, travel, and workforce analytics in enterprise organizations. The platform serves over 850 organizations and 2.2 million employees across 116+ countries.

## API

The Darwinbox API enables programmatic access to data stored in Darwinbox instances and supports importing data into the platform. API access is provided on a request-only basis to privileged users.

- **API Documentation**: https://api-docs.darwinbox.com/
- **Postman Collection**: https://www.postman.com/darwinbox-api/darwinbox-integration-team-s-public-workspace/

## API Categories

- Employee Management
- Attendance and Time Management
- Recruitment
- Organizational Management
- Payroll
- Background Verification
- Project Management
- Travel

## Authentication

Token-based authentication using SHA512 hashing of admin email + secret key + epoch timestamp. Also supports Basic Auth and OAuth 2.0.
